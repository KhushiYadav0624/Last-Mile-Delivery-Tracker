# Last Mile Delivery Tracker

## Overview
Last Mile Delivery Tracker is a MERN stack application for managing delivery operations. It allows admins to manage agents, zones, rate cards, and orders while customers can track deliveries.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express.js
- Database: MongoDB
- Deployment:
  - Frontend: Render
  - Backend: Render

## Features
- User Authentication
- Order Management
- Agent Management
- Zone Management
- Rate Card Management
- Order Tracking
- Dashboard

## Project Structure
```
backend/
frontend/
```

## Installation

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
